# Movie Review App

## Prereqs
* Node 12
* Yarn
* Angular 9
* Mongo DB

## Running the apps

1. Node Server
   1. In the root directory run `yarn`
   1. Next, run `node server.js`

1. Angular App
   1. In the public directory run `yarn`
   1. Next, run `ng serve`
   1. More instructions inside of the `./public/README.md`
